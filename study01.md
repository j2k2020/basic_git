## Git복습

```java
cd review //review 디렉토리로 이동
touch study01.md //study01.md파일을 생성
ls  //review 디렉토리 안의 모든 하위 폴더/파일 보기
code . //현재 파일 오픈
git status // git의 현재 상태 파악
git add [폴더/파일명.확장자] //커밋 전 add하기(커밋 전에 table에 올려 놓는다고 생각하기)
git commit -m "커밋 메모"  //add 해놓은 폴더/파일을 커밋(실행 확정상태로 만들기)
git push origin master //푸쉬 해서 올리기    

```
